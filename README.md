# Customer Churn Prediction and Sales Analysis

## 📌 Overview

This project predicts customer churn using Machine Learning and provides sales analysis insights through data visualization.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Streamlit

---

## 📊 Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Customer churn prediction
* Sales analysis visualization
* Machine Learning model evaluation
* Streamlit web application

---

## 🤖 Models Used

* Logistic Regression
* Random Forest Classifier
* Decision Tree Classifier

---

## ▶️ How to Run

### Install Libraries

```
pip install -r requirements.txt
```

### Run Jupyter Notebook

```
jupyter notebook
```

### Run Streamlit App

```
streamlit run streamlit_app.py
```

---

## 📈 Output

* Customer churn prediction
* Sales insights and visualizations
* Accuracy and performance analysis
